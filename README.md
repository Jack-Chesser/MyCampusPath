# MyCampusPath
Final project for my Rize Web Development course. 


1. Home Page (index.html)
- Shows the weekly schedule  
- Links to the Events page and Directions page  
- Has a working High Contrast accessibility mode  
- Uses Bootstrap for layout and styling  

2. Campus Events (page2.html)
- Shows featured events  
- Lets students submit their own event requests  
- Saves requests using `localStorage` so they stay even after refresh  
- Includes delete buttons for removing requests  

3. Directions & Weather (page3.html)
- Interactive campus map using Leaflet.js  
- Select two buildings to see:
  - Distance (in feet)  
  - Estimated walk time  
  - Current Jacksonville weather  
- Weather pulled live from the Open‑Meteo API  
- Building database stored in `buildings.js`  

Tech Used
- HTML / CSS / JavaScript**
- Bootstrap 5 (layout + buttons)
- Leaflet.js (interactive map)
- Open‑Meteo API (weather)
- LocalStorage (saving event requests)
- Custom IC theme (colors + cards)

Accessibility Features
- High Contrast Mode (black/yellow)
- Toggle button on every page  
- Mode stays saved using `localStorage`  
- Clean headings and simple layout  
- Good color contrast  
- Keyboard‑friendly
